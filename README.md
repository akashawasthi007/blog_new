## Installation

Install my-project with django

```bash
  pip install django
  pip install freeze
  pip install six
  cd blogging_website
  python manage.py runserver
```
   For creating superuser
   ```
   python manage.py createsuperuser
   enter name,email & password
   python manage.py makemigrations
   python manage.py migrate
   ```
